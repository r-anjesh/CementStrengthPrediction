How to Use Cement Strength Prediction

Welcome to the Cement Strength Prediction system! This tool uses advanced machine learning techniques to predict cement strength based on the mixture components. All the processing happens directly on this page. Follow the steps below:

Step 1: Input the Cement Data

Provide the absolute file path to your CSV file containing the cement mixture data. Your CSV should include the following columns:

  • Cement (kg in a m3 mixture)
  • Blast Furnace Slag (kg in a m3 mixture)
  • Fly Ash (kg in a m3 mixture)
  • Water (kg in a m3 mixture)
  • Superplasticizer (kg in a m3 mixture)
  • Coarse Aggregate (kg in a m3 mixture)
  • Fine Aggregate (kg in a m3 mixture)
  • Age (Days)
  • Concrete Compressive Strength (MPa)

Example of an absolute file path for a CSV: 

C:/Users/YourUsername/Documents/cement_mixture_data.csv

Step 2: Go to Home Page

After entering the file path, you will find the "Custom File Predict" button. Click on that button to start the prediction process, and the system will process the data and predict the compressive strength of the cement mixture.

Go to Home Page

Step 3: Review the Results

Once the prediction is complete, the results will be saved in the file Prediction_Output_File/Predictions.csv.

You can access the predicted cement compressive strength for your mixture in this file.
